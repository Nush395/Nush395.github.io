These are some of the projects I've tried out for fun.

  • Understanding TrueSkill

    I was interested in ranking players and Bayesian Machine Learning and implementing the original TrueSkill paper was a really fun way to do both.

    TrueSkill blog post
    I had to learn a lot of new stuff to understand the paper such as factor graphs and expectation propogation and how useful Gaussians are! One acid test for me to see if I've learnt something is to try explain it to someone else so I wrote an accompanying blog post explaining the theory and my motivations.
    Python implementation of TrueSkill
    I also implemented the paper in Python, please check it out, usual instructions in the README for usage.

  • Analog RNN

    I'm always excited by seeing articles that tie together ML and Physics. This paper formulates the wave equation using RNN update equations and shows how you could design a manifold with a non-linear wave speed across it to classify vowels by measuring the energy at points on the manifold!

    Python implementation of Analog RNN - WIP
    I wanted to implement the method the authors talk about by building a custom model in Tensorflow.
